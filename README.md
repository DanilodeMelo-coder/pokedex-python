Pokedex Python (Projeto de Estudo)

Este projeto é uma Pokedex simples feita em Python, criada como exercício prático para iniciar os estudos em Programação Orientada a Objetos (POO).

A aplicação funciona via terminal, permitindo adicionar e listar pokémons.

-- Objetivo do projeto

Praticar conceitos fundamentais como:

Classes

Métodos

Organização de código

Fluxo de execução

Estrutura de projetos Python

-- Funcionalidades atuais

Adicionar pokémons

Listar pokémons

Menu interativo no terminal

-- Melhorias planejadas

Criar uma classe Pokemon 🗸

Buscar pokémon pelo nome 🗸

Exibir detalhes de um pokémon específico 🗸

Separar o projeto em múltiplos arquivos (pokemon.py, pokedex.py, main.py)

Melhorar a organização do código

Adicionar persistência de dados futuramente (Json)
